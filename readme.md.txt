# STOCKMANAGERAPP · GitHub Pages (MVP)

This repo hosts the **front‑end** of STOCKMANAGERAPP. The app posts to a Google Apps Script backend Web App.

## Deploy (GitHub Pages)
1. Create a repo named `STOCKMANAGERAPP`.
2. Upload `index.html` to the repo **root**. (Optional: add the lock artifacts below.)
3. Commit to `main`.
4. In GitHub: **Settings → Pages → Source: Deploy from a branch → Branch: `main` `/ (root)` → Save**.
5. Open: `https://<your-username>.github.io/STOCKMANAGERAPP/`
