Risona Inc – Stock Manager v4

Pick open POs on Receiving. Browse and allocate customer invoices on Shipping. Create cycle-count cartons and print labels — all without leaving this screen.

📦 Receiving

Load open POs from your endpoint, pick a shipment from the list, select a line, and receive cartons directly into the registry.

🚚 Shipping

Load orders by ship date, pick an invoice, run allocation, review allocated cartons, mark ready-to-pack, and ship.

📋 Count / Putaway

Scan and verify cartons, move stock to storage locations, and create first-touch cycle-count carton labels.

🏷️ Labels

Load carton labels by date or carton ID, or accept a newly created cycle-count carton and print without leaving the app.

🔗 One endpoint

Paste your Apps Script URL once in the top bar — every tab uses the same endpoint. It is remembered on this browser.

📊 Live metrics

Each tab shows live KPIs pulled from the carton registry and order sheets so the floor always sees current status.
