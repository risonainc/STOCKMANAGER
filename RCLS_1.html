▶ RawPrintServer setup instructions

One-time setup on the warehouse PC (Windows):

  1. Download RawPrintServer.exe from SourceForge and copy it somewhere permanent (e.g. C:\Tools\).
  2. Open Command Prompt as Administrator.
  3. Run: RawPrintServer INSTALL "Zebra LP2844" 9100
    (Use the exact printer name shown in Windows Devices & Printers)
  4. Open Windows Defender Firewall → Advanced → Inbound Rules → New Rule: TCP port 9100, allow on Private network.
  5. Find the PC's local IP: open Command Prompt, type ipconfig, note the IPv4 address (e.g. 192.168.1.42).
  6. Enter that address in the field above: http://192.168.1.42:9100
  7. Click Test connection — if green, you're ready to print.

The service starts automatically with Windows. To remove it later: RawPrintServer REMOVE 9100

▶ About the EPL2 label format

Your UPS LP2844 (Part #120765-001) uses EPL2 (Eltron Programming Language). This app generates a complete EPL2 command string per label including:

  • CODE128 barcode — UPC or Carton ID
  • QR code — rendered as a 1-bit bitmap via the EPL2 GW (Graphic Write) command, since this printer model does not have the native Asian QR font set
  • Description, Brand, Location, Carton ID text fields
  • Qty, Dimensions, Weight metadata

The EPL2 data is sent as raw bytes — RawPrintServer passes it straight to the printer driver with no translation.
